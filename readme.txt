== Description ==

WordPress the new post will be automatically sent to the Follow5.com
If you have any question or find any bug, you can visit [Plugin Homepage](http://immmmm.com/wp-follow5.html "WP-Follow5") to submit it.

将WordPress新日志自动发送至 Follow5.com

== Installation ==

It's very easy to install the plugin.

1. Upload the whole directory to "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress.
3. That's all.
